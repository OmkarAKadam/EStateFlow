# User Stories

All statements are written from the perspective of system users.

---

## Owner (Property Lister)

### Create

- User can create an account.
- User can log in to their account.
- User can create a property listing.
- User can upload images for a property.
- User can respond to messages regarding their property.

### View

- User can view their profile information.
- User can view all properties they have listed.
- User can view full details of any property.
- User can view messages received from other users.

### Update

- User can update their profile information.
- User can edit details of a property they created.
- User can update property images.

### Delete

- User can delete a property they created.
- User can delete their own account.

### Authentication Required

- User must log in to create a property.
- User must log in to edit a property.
- User must log in to delete a property.
- User must log in to view received messages.
- User must log in to manage their listings.

---

## Tenant (Viewer)

### Create

- User can create an account.
- User can log in to their account.
- User can save a property to favorites.
- User can send a message to a property owner.

### View

- User can view all available properties.
- User can filter properties by price.
- User can filter properties by property type.
- User can view full property details.
- User can view properties they saved.
- User can view messages they sent.

### Update

- User can update their profile information.
- User can remove a property from favorites.

### Delete

- User can delete a saved property from favorites.
- User can delete their own account.

### Authentication Required

- User must log in to save a property.
- User must log in to send a message to an owner.
- User must log in to view saved properties.
- User must log in to view sent messages.

---

## Admin (Basic System Oversight)

### Create

- Admin can create an account.

### View

- Admin can view all users.
- Admin can view all properties.

### Update

- Admin can disable a user account.
- Admin can enable a user account.

### Delete

- Admin can delete any property.
- Admin can delete any user account.
